# **프로젝트 간단 소개(BillBoard)**
-본 프로젝트(앱은) 사용자가 회원가입을 하여 퀵보드를 등록하고 대여를 모방하여 만든 프로젝트입니다.  
주된 기능으로는 SMTP를 이용한 회원가입, 퀵보드 등록,퀵보드 승하차시 로그 기록, 주소 검색하여 맵 이동 등이 구현되어있습니다.
